Özellikler

Underline Tabs Clean Accordion Standard Feed
  • Klasik underline tab göstergesi — Chrome DevTools ve VS Code'dan tanıdık
  • Temiz accordion: chevron + başlık + badge. Minimal, net
  • Feed satırları: method badge → URL → eşleşme etiketi. Tek satır, tek bakış
  • HAR yüklü durumda dosya bilgisi + endpoint sayısı + mode toggle bir arada
  • En güvenli seçim — developer'ların alışık olduğu pattern'lar

Direction 4: Dense Feed + Expanded Row

Daha yoğun feed satırları ve tıklanınca açılan response detay paneli. Developer Tools'un Network tab'ı gibi.

GET /api/users/{id}/settings HAR ✓
GET /api/billing/status Rule ✓
PUT /api/users/{id}/preferences HAR ✓

Özellikler

Dense Feed Expanded Row Inline JSON Editor
  • Daha sıkışık feed satırları (6px padding) — daha fazla satır görünür
  • Satıra tıklayınca response detayı INLINE açılır — ayrı bir panel yok
  • JSON editör syntax highlighting ile — key, string, boolean renk kodlu
  • "Save" butonu response'u HAR session'ına persist eder
  • Chrome DevTools Network tab'ının kompakt hissini yakalamaya çalışır
  • Daha fazla bilgi, daha az alan — information density yüksek

Özellikler

Card Layout Border Separation Status Indicator
  • Her bölüm bağımsız bir kart — border + radius ile net görsel ayrım
  • Kart başlığında section adı + badge + status göstergesi
  • "Aktif" yeşil dot ile intercept durumu — VS Code status bar mantığı
  • Arka plan farklı tonda (gri) — kartlar beyaz üzerinde float ediyor
  • Dezavantaj: Daha fazla padding = daha az içerik alanı. Kompakt popup'ta alan kaybı
  • Avantaj: Her bölüm çok net ayrılıyor — bilgi grupları belirgin

Özellikler

Full Rule Form Exclude List Multi-Accordion
  • Rule formu: URL pattern input + method dropdown + status code + response body + delay
  • Mevcut rule'lar liste halinde — method badge + pattern + status + silme butonu
  • Settings bölümünde Extension toggle + exclude list birlikte
  • Exclude listesinde URL pattern + × silme — basit ve net
  • Birden fazla accordion aynı anda açık — developer ihtiyaç duyduğu bölümleri görebiliyor
  • Bu direction Journey 3'ü (rule-based error test) canlı gösteriyor
